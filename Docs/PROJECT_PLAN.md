# 《无限求生》项目计划（V1）

## 1. 定位

- 类型：单机第三人称俯视角求生 RPG。
- 架构：通用求生底层 + 多版本世界包。
- 首发版本：公路求生（RoadSurvival）。

## 2. 核心玩法循环

进入世界 -> 收集资源 -> 制作/建造 -> 抵御危险 -> 事件选择 -> 阶段推进 -> 结局。

## 3. 开发阶段

### 阶段 A：核心操作层
- 角色移动
- 俯视镜头跟随
- 交互与采集
- 基础战斗

### 阶段 B：生存系统层
- 生命/饥饿/口渴
- 时间与天气
- 背包与物品使用
- 制作系统

### 阶段 C：版本系统层
- 版本选择 UI
- 配置装载器
- 版本差异（属性、资源、危险、胜利条件）

### 阶段 D：公路求生 Demo
- 车辆状态（耐久/燃油/载重）
- 节点地图推进
- 事件系统（10 条）
- 车辆升级（至少 2 档）

## 4. 版权与内容规范

- 不直接引用现有小说文本、角色名、专有道具名。
- 仅采用“类型化灵感”，保证原创设定。

## 5. 里程碑

- M1：可跑通角色移动与采集。
- M2：可跑通背包 + 制作 + 基础战斗。
- M3：可跑通 RoadSurvival 节点推进 + 事件。
- M4：可达成首个结局（安全营地）。
